## Prerequisites

- node ^16

## Install
Make sure that java install on your machine

install firebase cli
```sh
npm i -g firebase-tools
```

init emulators
```shell
firebase init emulators
```

install dependencies

```sh
yarn install
```

## Usage

```sh
yarn start
```
